##操作方法
Eキーでオープニングの文字送り、ゲーム画面での調べる
スペースキーでお札を発射
お札が当たると一定時間敵が動けなくなります

敵をかわしつつ、鍵を探し、正解の扉を見つけて
ボスを倒して謎の空間から脱出しましょう
